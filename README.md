## ğŸ§ What is Daily Question?

**Daily Question** is a minimalist, offline-first journaling application designed for the modern web. It strips away the clutter of complex productivity tools, offering you a single, thoughtful prompt every day. 

Built entirely in a **single HTML file**, it requires no installation, no database, and no internet connection. Your thoughts remain 100% private, stored securely on your own device.
